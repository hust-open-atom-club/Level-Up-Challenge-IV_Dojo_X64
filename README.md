# Spring 2026 - 综合实践 IV - Linux 内核攻防实践

本道馆为网安学院学生打造的实战训练场景，聚焦 Linux 内核漏洞挖掘、防御加固与漏洞利用对抗。通过还原真实漏洞场景及内核调试工具链，提供内核提权、安全机制绕过等靶场，构建从漏洞分析到攻防对抗的闭环训练体系。

## 模块与关卡

### 漏洞利用（kernel-exploitation）

|关卡|难度|技术要点|
|------|------|----------|
|vm|Level 2|QEMU 虚拟机环境入门，`vm` 命令使用，内核模块交互|
|nullptr|Level 3|空指针解引用基础：`mmap_min_addr=0`，零地址映射提权|
|nullptr2|Level 4|空指针解引用进阶：`mmap_min_addr=65536`，绕过限制提权|
|snow|Level 5|释放后使用（UAF）：SLUB 堆风水占位，控制流劫持|
|maze|Level 6|脏管道（CVE-2022-0847）：UAF 篡改 pipe_buffer 元数据覆写只读文件|
